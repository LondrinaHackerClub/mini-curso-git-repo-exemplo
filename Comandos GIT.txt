
Principais Comandos

git clone - Clona um repositório (snapshot do estado atual do repositório)

git init - Inicia um novo repositório

git add {NOME_DO_ARQUIVO} - Adiciona arquivo ao controle de versão

git add . - Adiciona todos os arquivos ao controle de versão

git remote add {NOME_DESTINO} {ENDEREÇO_DESTINO} - adiciona um endereço remoto

git push -u {NOME_DESTINO} {nome_da_branch} - empurra as alterações locais para o repositório e branch de destino

git commit -m "MENSAGEM"- Realiza o commit de uma alteração com comentários


git pull - puxa alterações na branch atual 

git fetch - puxa alterações de Index das branchs

git checkout {nome_da_branch} - altera a branch atual

git checkout -b {nome_da_branch} - altera a branch atual (cria se ela não existir)

git merge - Mescla duas branchs (esse é que da merda)
